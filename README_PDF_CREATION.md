# Как создать идеальный PDF из бизнес-плана

## Готовые файлы:
- `ZENO_CY_BusinessPlan_Print_Ready.html` - оптимизированная версия для печати
- `print-styles.css` - профессиональные стили

## Рекомендуемые способы создания PDF:

### 1. Через браузер (ЛУЧШИЙ способ):
1. Открой `ZENO_CY_BusinessPlan_Print_Ready.html` в браузере
2. Нажми `Cmd+P` (печать)
3. Выбери "Сохранить как PDF"
4. Настройки:
   - Размер: A4
   - Поля: Обычные
   - Масштаб: 100%
   - Включить фоновую графику: ДА

### 2. Через Typora (если установлен):
```bash
# Установи Typora, затем:
typora ZENO_CY_BusinessPlan_FULL_UPDATED.md --export pdf
```

### 3. Через Markdown editors:
- Mark Text
- MacDown  
- Obsidian (с плагином)

### 4. Онлайн конвертеры:
- markdown-pdf.com
- pandoc.org/try

## Настройки для идеального PDF:
- Разрывы страниц: автоматические
- Шрифт: системный (11pt)
- Поля: 2.5см сверху, 2см остальные
- Нумерация страниц: включена
- Оглавление: автоматическое

## Результат:
Профессиональный PDF готовый для презентации инвесторам.
